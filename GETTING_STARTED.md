# Getting Started Guide 🎮

## Welcome to Evergreen Adventure!

This guide will help you get started on your adventure in Evergreen Valley.

## Installation

1. **Download the game files**
   ```bash
   git clone https://github.com/willishim-pixel/LALALA.git
   cd LALALA
   ```

2. **Open in browser**
   - Simply open `index.html` in your web browser
   - For best results, use Chrome, Firefox, Safari, or Edge
   - Mobile: Open on iPhone Safari or Chrome Android

3. **Local Server (Optional)**
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Using Python 2
   python -m SimpleHTTPServer 8000
   
   # Using Node.js
   npx http-server
   ```
   Then visit `http://localhost:8000`

## First Time Playing

### Step 1: Explore the World 🌍
- Use **Arrow Keys** or **WASD** to move
- The world is divided into 8 regions, each with unique locations
- Press **E** or **Enter** to interact with NPCs and buildings

### Step 2: Get a Job 💼
1. Click the **📱 Smartphone button** (bottom right)
2. Open the **📋 Jobs** app
3. Browse available careers
4. Click on a job to apply
5. Start working and earning money!

### Step 3: Build Relationships 👥
1. Open **Smartphone → Contacts**
2. Click on an NPC to talk to them
3. Choose dialogue options to build relationships
4. Give them gifts to increase friendship
5. Different NPCs like different items

### Step 4: Buy a Home 🏠
1. Earn $15,000+
2. Open **Smartphone → Banking** to check your balance
3. Explore real estate locations
4. Purchase a home (prices range from $10,000-$100,000)
5. Customize your new home with furniture

### Step 5: Get a Vehicle 🚗
1. Earn some money ($300+)
2. Visit vehicle dealerships
3. Purchase your first vehicle
4. Use it for fast travel across the world

## Game Systems Explained

### Money & Banking 💰
- **Starting Money**: $500
- **Earning**: Complete jobs and activities
- **Spending**: Buy homes, vehicles, items
- **Saving**: Use Banking app to check balance

### Stats & Health ❤️
- **Energy**: Decreases when moving, resets with sleep
- **Mood**: Affected by activities and social interactions
- **Hunger**: Increases over time, restore with food

## Troubleshooting

### Game Won't Start
- Make sure JavaScript is enabled in your browser
- Try a different browser
- Clear browser cache
- Check browser console for errors (F12)

### Lag or Performance Issues
- Lower your graphics quality setting
- Close unnecessary browser tabs
- Update your browser
- Try fullscreen mode
- On mobile: close background apps

### Save Data Lost
- Game auto-saves every 5 minutes
- Check if localStorage is enabled
- Try a different browser (may have different storage)
- Game saves in browser local storage, not cloud

### Controls Not Working
- Make sure the game window is focused
- Try different key combinations
- On mobile: make sure touchscreen is working
- Check accessibility settings
